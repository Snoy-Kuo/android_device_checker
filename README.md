# android_device_checker

An Android example App that shows some device and resettable ids includes:
 - UUID
 - SSAID
 - AAID
 - IMEI
 - MAC address
 - IP address
 - Public IP Address

## Dev env

 - macOS 11.6 (Big Sur) x64
 - Android Studio Bumblebee Patch 1
 - Android SDK version 31
 - JDK: 11
 - Gradle: 7.2
 - Kotlin: 1.6.10
